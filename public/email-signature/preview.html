How to install in Gmail

  1. Deploy the website so https://farhand.live/email-signature/* is live (Vercel auto-deploy handles this).
  2. Open https://farhand.live/email-signature/signature.html in your browser.
  3. Select the rendered signature block (not the HTML source) and copy.
  4. Gmail → Settings (gear) → See all settings → General → Signature.
  5. Paste into a new signature. Save changes.

How to install in Apple Mail

  1. Open https://farhand.live/email-signature/signature.html in Safari.
  2. Select the rendered signature → Edit → Copy.
  3. Apple Mail → Settings → Signatures → + new.
  4. Uncheck "Always match my default message font".
  5. Paste the signature. Close. Choose it as the default for aaryan@farhand.live.
